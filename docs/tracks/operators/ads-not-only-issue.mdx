## 1. 认知转变

广告是放大器，不是魔法。投放把现有体验放大：好体验放大转化，坏体验放大亏损。

## 2. 系统视角的四个环节

- PDP：价值/风险/决策加速信息是否清晰。
- Cart：成本透明、信任、修改便利、推荐不打扰。
- Checkout：摩擦最小、支付信任、表单精简。
- 信任机制：物流/退换/客服/评价是否可见。

## 3. 常见非投放原因

- 首屏看不出买点；价格/优惠/库存不透明；物流/退换信息缺失。
- 变体/数量修改困难；Cart/Checkout 成本突然上升；移动端遮挡/卡顿。
- 埋点缺失，不知道哪里掉。

## 4. 快速修复思路（运营可控）

- 在 PDP/Cart 显示总价/优惠/运费估算，减少“结账才发现”。
- 补齐信任信息：配送/退换/客服入口同屏可见。
- 精简 Checkout：去掉非必要字段，支持自动填充/访客结账。
- 移动端巡检：按钮可见、键盘不遮挡、加载可接受。
- 埋点：首屏曝光、变体切换、加购、结账开始/完成。

## 5. Checklist

- [ ] 核心卖点、价格/优惠、CTA 首屏可见。
- [ ] Cart 展示总价/运费/税费说明，修改便捷。
- [ ] Checkout 表单最小化，支付方式符合市场习惯。
- [ ] 信任信息在 PDP/Cart/Checkout 同屏露出。
- [ ] 埋点可用，能看到每步流失；移动端体验通过自测。
