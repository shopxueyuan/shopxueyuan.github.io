## 1. 为什么很多 PDP 看起来好看，但不转化

- 只优化视觉，没有回答“为什么买”：卖点、证据、场景缺失，用户看完仍然犹豫。
- 信息顺序混乱：先看到无关内容，关键信息被折叠或埋在底部。
- 缺乏行动引导：CTA 不够明确，缺少配送/退换等信任信息，用户无法下定决心。

## 2. PDP 的用户决策路径拆解

1) **我看到的是什么？**（基础信息、品牌、核心卖点）  
2) **适合我吗？**（规格、尺寸、适用场景、SKU 匹配）  
3) **值不值？**（价格、优惠、对比、社会证明）  
4) **能不能信？**（物流时效、退换保障、支付安全）  
5) **怎么下单？**（清晰 CTA、加购/直购、附加选项）  

模块排布与信息露出顺序应贴合这个路径，而不是随意堆叠素材。

## 3. 高转化 PDP 的核心模块

- **商品价值**：主视觉 + 卖点摘要（3 条以内），关键规格/适用场景。让用户 3 秒内知道“买它的理由”。
- **风险消除**：物流时效、退换政策、质保、支付安全、真实评价与问答。降低“买了会亏”的担心。
- **决策加速**：清晰的变体选择、优惠展示、分期/赠品提示、推荐搭配与套餐。让用户少纠结、快下单。

## 4. 不同行业 PDP 模块差异

- **服饰/鞋包**：尺码助手、真人实拍、材质护理、试穿反馈、搭配推荐。  
- **美妆个护**: 成分/功效、适用肤质、敏感肌说明、使用步骤与频次、功效对比图。  
- **家居/3C**: 尺寸/参数表、安装/兼容性、质保/售后、视频演示、配件清单。  
- **食品/营养**: 成分表、过敏原、保质期、产地/溯源、食用方法。  

行业不同，核心逻辑相同：先价值，再证据，再风险消除，再引导行动。

## 5. 常见运营误区

- 以素材堆砌代替结构思考：大量长图和故事，但缺少关键信息与 CTA。  
- 把优惠信息藏得很深：用户看不到“现在买有什么好处”，错失加速机会。  
- 忽视信任模块：不写退换、物流、保障，或把它们放到底部，导致犹豫退出。  
- 模块顺序随意：核心卖点和规格被社交证明、长图挤到下面，打乱决策路径。  

## 6. 一个标准 PDP 模块清单（Checklist）

- [ ] 首屏：主视觉 + 核心卖点摘要（<=3 条）+ 清晰 CTA。  
- [ ] 变体选择：易懂的文案/色块/尺寸，默认选中，错误提示清晰。  
- [ ] 价格与优惠：当前价、划线价/优惠、分期/赠品提示，同屏可见。  
- [ ] 规格参数：关键规格表（可折叠），避免长段文字。  
- [ ] 使用/场景：短视频或图文，展示真实使用与适用人群。  
- [ ] 信任信息：物流时效、退换/质保、支付安全、客服入口。  
- [ ] 社会证明：评价/问答精选，真实场景的图片/视频。  
- [ ] 推荐与搭配：同系列/套餐/Upsell，避免打断主路径。  
- [ ] 页内 CTA 重复：长页中部/底部有二次 CTA，减少滚动后的回溯成本。
