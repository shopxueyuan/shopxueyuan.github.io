## 为什么这是最重要的决策之一

选错实现形态，往往意味着：
- 后期无法扩展
- 技术债迅速堆积
- 平台审核受阻

---

## 三种形态的本质区别

| 形态 | 本质 |
| --- | --- |
| Theme | 前端展示与交互 |
| Public App | 可复用的第三方能力 |
| Custom App | 商家私有系统集成 |

---

## 什么时候用 Theme？

适合：
- 页面结构
- 交互逻辑
- 视觉呈现

不适合：
- 跨系统业务
- 持久化规则

---

## 什么时候用 Public App？

适合：
- 多商家复用
- SaaS 能力输出

不适合：
- 高度定制逻辑

---

## 什么时候用 Custom App？

适合：
- 企业私有流程
- ERP / CRM / BI 集成
- 自动化规则

---

## 决策速查表

| 需求 | 推荐 |
| --- | --- |
| 页面效果 | Theme |
| 功能扩展 | Public App |
| 企业系统 | Custom App |
