## 1. 先看资源与目标

- 资源：供应链广度/深度、资金、运营人力、内容产能。
- 目标：现金流速度 vs 品牌资产；短周期套利 vs 长期复购。

## 2. 铺货模式（优劣）

- 优势：上新快、测试需求广、爆品可能性高。
- 劣势：库存/物流复杂、客服压力、复购低、评价风险高。
- 适合：高选品能力、可接受高试错、对品牌一致性要求低。

## 3. 精品模式（优劣）

- 优势：品牌一致性、复购/溢价、运营可聚焦、供应链可控。
- 劣势：前期转化慢、对内容/研发要求高、爆款概率低但单品深。
- 适合：有差异化产品力、内容/品牌能力、愿意长期经营。

## 4. 自我评估 Checklist

- [ ] 供应链：能否稳定提供少数高质量产品？还是广泛但浅。
- [ ] 资金周转：能撑多久的内容/品牌投入？能否承受库存风险。
- [ ] 团队能力：有无内容制作/品牌运营？有无选品/采购能力。
- [ ] 市场策略：追求短期现金流还是长期品牌资产。
- [ ] 客单与复购：品类是否天然高复购？是否需要教育/内容。
- [ ] 售后/客服：能否支撑大量 SKU 的问题与退换？

## 5. 决策建议

- 资源有限且不确定选品：先“小批量铺货 + 数据验证”，逐步收敛为半精品。
- 有差异化产品力：直接走精品路线，聚焦 1-3 个产品线，内容/信任建设优先。
- 无论模式：都需要成本/毛利核算、供应链与客服准备，避免“卖出去才发现亏”。
