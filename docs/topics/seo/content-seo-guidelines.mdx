# 内容页 SEO 规范模板

## Title 命名规则
- 公式：`【主题/场景】 + 【关键动作/方法】 + Shopify`，必要时补充角色前缀（如“运营视角”/“开发实践”）。
- 避免无意义的长尾堆砌，保持可读性与可扫描。

## Description 写作规则
- 80–160 字，首句点明场景与受众，自然出现 Shopify 与核心动作。
- 说明解决的问题/决策依据，避免营销口吻。
- 每页唯一，不复用。

## 关键词分层策略
- 主关键词：Shopify + 核心主题（如「Shopify 转化优化」「Shopify API 集成」）。
- 角色关键词：商家 / 运营 / 开发 / 企业 / 决策者。
- 场景关键词：选品、转化率、A/B 测试、Webhook、架构、成本、合规等。

## 示例
- 示例 1  
  - Title：Shopify 转化率优化清单｜运营视角的落地方法  
  - Description：面向运营的 Shopify 转化率优化指南，梳理首页/产品页/结算页的关键要点与常见实验思路，帮助团队在不依赖大量开发的情况下提升转化。  
  - 关键词：Shopify 转化优化；运营；A/B 测试；漏斗
- 示例 2  
  - Title：Shopify Storefront API 与 Hydrogen 性能优化｜开发实践  
  - Description：针对开发者的 Storefront API 与 Hydrogen 性能优化实践，涵盖缓存、延迟加载、监控与 CI/CD 集成，确保前后端在增长迭代中保持稳定体验。  
  - 关键词：Shopify API；开发；性能优化；监控
