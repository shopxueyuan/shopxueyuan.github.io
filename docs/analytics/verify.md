---
id: verify
title: 埋点验证清单
---

1. 本地验证：在浏览器开启网络面板与 console，确认事件 payload 字段完整、无异常 4xx/5xx。
2. 预发验证：在沙箱店铺或预览主题上，以真实流程（浏览-加购-下单）跑一遍，记录事件流水。
3. 监控：
   - 使用百度统计实时查看 PV/UV 与事件触发。
   - 如有埋点网关，使用日志查找特定 `session_id` 以确认落库。
4. 回归：每次主题/APP 发布后，抽样验证关键路径事件是否仍然上报。

> 提示：为便于回溯，可在事件中加入 `release_version` 或 Git `commit` 字段。
