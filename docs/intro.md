欢迎来到 **Shopify 学院**。这里聚焦开发者视角（Developer Track），用简体中文整理可复用的实践、规范与示例，帮助你快速搭建、上线并持续优化 Shopify 站点。

## 你能在这里找到什么
- 入门：本地环境、CLI 与账户的最小可行配置。
- Theme 工程化：如何把主题开发纳入团队工程体系（版本、协作、质量）。
- App / API：构建可安装的 App 与使用 Storefront/Admin API 的最佳实践。
- 埋点 / 数据：埋点规范、验证与可观测性。
- Recipes：常用代码片段与集成示例，拿来即用。

## 学习方式
- 从左侧导航按模块学习，也可以直接跳到 Recipes 找到对应代码段。
- 所有内容使用 Markdown/MDX，你可直接提交 PR 丰富文档。

准备好后，先完成「入门」章节的环境准备吧。
